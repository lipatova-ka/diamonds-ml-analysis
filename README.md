# Анализ данных алмазного рынка

Репозиторий содержит программную реализацию,
разработанную в рамках выпускной квалификационной работы
«Применение методов классификации и кластеризации в анализе данных алмазного рынка».

## Используемые технологии
- Python
- pandas
- numpy
- scikit-learn
- matplotlib

## Структура проекта
- src/main.py — формирование сбалансированной выборки, используемой на последующих этапах анализа
- src/data_cleaning.py — очистка и предварительная обработка данных
- src/eda_chapter_3.py — разведочный анализ
- src/additional_characteristic.py — анализ дополнительных характеристик (Fluor, Symmetry, Polish)
- src/classification.py — модели классификации
- src/clustering.py — методы кластеризации

## Примечание
Полный датасет не публикуется в связи с его объёмом и ограничениями источника данных.
Файл, содержащий датасет, используемый для анализа на 50к строк приложен в виде архивного файла из-за органичений размера файла.
