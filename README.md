# Анализ данных алмазного рынка

Репозиторий содержит программную реализацию,
разработанную в рамках выпускной квалификационной работы
«Применение методов классификации и кластеризации в анализе данных алмазного рынка».

## Используемые технологии
- Python
- pandas
- numpy
- scikit-learn
- matplotlib

## Структура проекта
- src/eda.py — разведочный анализ данных
- src/preprocessing.py — предобработка данных
- src/classification.py — модели классификации
- src/clustering.py — методы кластеризации

## Примечание
Полный датасет не публикуется в связи с его объёмом и ограничениями источника данных.
